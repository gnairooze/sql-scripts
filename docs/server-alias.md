# Add SQL Server Alias

1. browse `c:\windows\system32`
2. run `cliconfg` "SQL Server Client Network Utility"
![[2023-03-02.png]]
4. select "Alias" tab
![[2023-03-02 (1).png]]
5. click "Add button"
![[2023-03-02 (2).png]]
6. write "Server alias"
7. select "TCP/IP" in Network libraries
8. write "Server name"
